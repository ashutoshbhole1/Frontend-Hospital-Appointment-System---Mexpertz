# 🏥 Hospital Appointment System - Frontend

![Banner](./src/assets/readme image hero section.png)

## ✨ Overview
A modern, intuitive, and responsive web application designed to streamline the process of booking hospital appointments. Built with speed and user experience in mind, this platform connects patients with doctors effortlessly.

---

## 🚀 Key Features

- **🔐 Secure Authentication** - Robust login and registration for patients and doctors.
- **📅 Smart Scheduling** - Real-time availability check and instant booking.
- **👨‍⚕️ Doctor Discovery** - Search and filter doctors by department and expertise.
- **📱 Responsive Design** - Seamless experience across desktops, tablets, and mobile devices.
- **📊 User Dashboard** - Comprehensive view of upcoming and past appointments.
- **🔔 Real-time Notifications** - Stay updated with appointment status changes.

## 🏁 Getting Started

### Prerequisites
- Node.js (v16.x or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory**
   ```bash
   cd "Frontend Hospital Appointment System - Mexpertz"
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Set up Environment Variables**
   Create a `.env` file in the root and add:
   ```env
   VITE_API_URL=http://localhost:5000/api
   ```

5. **Run the development server**
   ```bash
   npm run dev
   ```
